FLINT-UI Inputs
=========================

We currently use five main data-types that are used as FLINT inputs - geoTIFF, JSON, CSV, SQL DB, Shapefile. We have hosted the python notebooks for each data type that contains the instructions and the required dependencies to view each file on a python notebook.
